# MOD to MP4 converter via FFmpeg

Simple program for converting .MOD files into mp4 with high quality.
Support .MOI files
Automaticly download ffmpeg static binary for corresponding platform

```bash
Usage: mod2mp4-rs [OPTIONS] --output <OUTPUT> <--file <FILE>|--dir <DIR>>

Options:
  -f, --file <FILE>
  -d, --dir <DIR>
  -o, --output <OUTPUT>
      --date
  -h, --help             Print help
```
